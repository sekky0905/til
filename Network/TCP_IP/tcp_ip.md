# TCP/IPについて

- エフェメラルポート(ephemeral port)
    - TCP/IPのポート番号は、サーバにもクラインアントにも存在する
        - クライアントには、未使用のものがランダムで割り当てられ、これがエフェメラルポート(ephemeral port)と呼ばれる
            - サーバと接続している時だけ割り当てられる一時的なもの
